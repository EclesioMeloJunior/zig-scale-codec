# Zig SCALE Codec

Zig implementation of the SCALE (Simple Concatenated Aggregate Little-Endian)

- [x] Fixed Integers (i8, i16, i32, i64, i128, u8, u16, u32, u64, u128)
- [x] Compact Integers
- [x] Boolean
- [x] Options
- [x] Strings
- [x] Structs
- [ ] Results
- [ ] Vectors
- [ ] Tuples
- [ ] Enumerations (tagged-unions)
